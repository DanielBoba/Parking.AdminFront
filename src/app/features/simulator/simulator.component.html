Acerca del Simulador

Esta herramienta inyecta eventos de reconocimiento de patentes directamente en el Motor de Colas (RabbitMQ/MQTT) de la API principal.

Flujo de Ejecución:

  1. Se ensambla un payload JSON 100% idéntico al que la Jetson emitiría en vivo.
  2. El Backend intercepta y lo publica bajo el Tópico/Queue de tu Tenant.
  3. El servicio "ALPRParkingEngineWorker" escucha el mensaje de la cola.
  4. El flujo de autorización transcurre de modo real (Reglas de acceso).
  5. Podrás ver el cruce en la base de datos o en la pestaña Live de los Accesos.
